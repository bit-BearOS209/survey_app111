# Опросник: Использование заметок в облаке

Учебная практика 2026 года.

## Описание проекта
Веб-приложение на Python (Streamlit) для сбора анкет о том, как студенты используют облачные заметки, с сохранением данных в Firebase Firestore.

## Как запустить
1. Установить зависимости: `pip install -r requirements.txt`
2. Запустить приложение: `streamlit run app.py`
